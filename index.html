tetrominet by wjordan

Browser-based, multiplayer, open-source implementation of the popular tetromino puzzle game. Currently deployed on an Amazon S3 instance at tetromi.net!

TetromiNET is a browser-based, multiplayer, open-source implementation of the popular tetromino puzzle game. Featuring:

  • Written in Scala, a highly expressive language that compiles to JVM bytecode and is fully interoperable with existing Java libraries
  • Model-View-Controller pattern effectively separates the game logic, the server framework, and the front-end Java applet. A fully-functional text-mode interface is also included to demonstrate the model-view separation. (Hopefully a mobile phone port is coming soon!)
  • Extremely efficient XML-based client-server protocol, designed to cope with arbitrary latency between players, and completely eliminates any possibility of cheating or hacked clients. Clients capture and send compressed sequences of controller inputs, and the game is replayed exactly on both the opponent's client and the delegating server.
  • Uses the Lift web framework to implement the game server, also managing browser session state in order to match games to users. (A more full-featured game lobby is planned for the future!)
  • Currently deployed on an Amazon EC2 instance at tetromi.net!

Dependencies

  • Maven (project dependency management)
  • Scala (Programming language compiler and libraries)
  • Lift (Scala-based web framework)
  • PulpCore (2D Java applet framework)
  • Charva (Text-mode API bindings for Java)

Install

Compilation is done through the Maven framework. More details to come in the future. For now, just go to http://tetromi.net to try it out!
